# keychron-colemak-dh

- made for Mac with regular US qwerty keyboard 

- [Colemak mod DH as base](https://colemakmods.github.io/mod-dh/)

- Spacebar + rst for ( { [
- Spacebar + nei for ] } )

- Spacebar + g for \
- Spacebar + m for /

- Spacebar + c for ,
- Spacebar + h for .
- Spacebar + d for ;

- Caps + neio for left, up, down, right
- Caps + f for hyphen
- Caps + u for equal
